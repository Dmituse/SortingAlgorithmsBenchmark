# Sorting Algorithms Benchmark

Это консольное приложение для сравнения производительности алгоритмов сортировки на массивах разных типов и размеров. Пользователь задаёт параметры массива, данные копируются и сортируются разными алгоритмами с измерением времени выполнения. Позволяет наглядно оценить влияние структуры данных и объёма на скорость работы алгоритмов. Использует стандартный C++, без ручного управления памятью, легко расширяемо.
